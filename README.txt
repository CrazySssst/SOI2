# SOI2
钢铁之魂2 (by babylon.js)

0.学习资料
http://doc.babylonjs.com/

1.工程结构说明（杜鹏宇）
asset  游戏资源
css    css文件
doc    参考文档
js     javascript文件
lib    外部依赖项
page   游戏页面
temp   临时文件

2.坦克参数说明（杜鹏宇）
武器系统：影响攻击伤害、弹道速度
火控系统：影响发射冷却、炮台转速
动力系统：影响移动速度
装甲系统：影响防御减伤
雷达系统：影响侦察范围（暂不开发）
附加系统：特殊武器（暂不开发）

3.坦克类型说明（杜鹏宇）
侦察战车：高行动力，低战斗力
重型坦克：各项性能均衡
自行火炮：高杀伤力，低行动力

4.第一版目标（杜鹏宇）
玩家使用战场名称建立游戏或加入已存在游戏，
建立游戏的玩家在本地浏览器运行服务器逻辑，
新加入的玩家根据当前玩家数的奇偶自动分边，
能实现多人的坦克、子弹、地图逻辑与同步。
